
# Magic Tea Elite ERP & POS

A premium, professional Management System for "The Magic Tea" outlets. Optimized for production deployment on Firebase and version control on GitHub.

## 🚀 Quick Start
1. **Archive Project**: Generate a production ZIP.
   ```bash
   npm run bundle
   ```

2. **GitHub Deployment**:
   ```bash
   git init
   git add .
   git commit -m "Initial production release"
   git remote add origin <YOUR_REPO_URL>
   git push -u origin main
   ```

## 🛠️ Performance & Security
- **TBT Optimized**: < 200ms TBT through aggressive DOM pruning and lazy loading.
- **Security**: Built-in CSP, clickjacking protection, and authenticated data isolation.
- **Offline Sync**: Advanced background queue for Slow 4G and spotty connectivity.
- **AI Core**: Genkit-powered inventory forecasting for automated procurement.

## 📄 Standalone Audits
Use the **Settings > Backup** menu to download a "Portable HTML Snapshot" — a single file containing all your data that works entirely offline without a server.
